# Kaveri Traders — Premium Wood Supply Website

A premium scrollytelling portfolio website for **Kaveri Traders**, built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Next.js 16** (App Router, static export)
- **TypeScript**
- **Tailwind CSS v4**
- **Framer Motion** (scroll animations)
- **Lenis** (smooth scrolling)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Build

```bash
npm run build
```

Static output is generated in the `out/` directory.
